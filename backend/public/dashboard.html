Códigos de Activación

Haz clic en Generar código, luego sigue los pasos para activar tu reproductor.

Si el reproductor está conectado por cable
1. Conecta el reproductor a la energía y al televisor vía HDMI.
2. Conecta el cable de red y enciéndelo — el televisor mostrará el nombre de una red WiFi temporal.
3. Conecta tu celular a esa red WiFi. El nombre empieza por SCMS- (ej: SCMS-RO01AB). Contraseña: sonorocms
4. Abre http://10.42.0.1:8080 en el navegador de tu celular.
5. Ingresa el código de activación que generaste aquí.
Si el reproductor no tiene cable
1. Conecta el reproductor a la energía y al televisor vía HDMI.
2. Enciéndelo — el televisor mostrará el nombre de una red WiFi temporal.
3. Conecta tu celular a esa red WiFi. El nombre empieza por SCMS- (ej: SCMS-RO01AB). Contraseña: sonorocms
4. Abre http://10.42.0.1:8080 en el navegador de tu celular.
5. Selecciona tu red WiFi, ingresa la contraseña y el código de activación.
